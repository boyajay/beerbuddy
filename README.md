# Beer Buddy
Search for beer brands by their type, and add selections to wishlists and favorites.

## Team

  - __Product Owner, Web Developer__: Boya Jiang
  
## Requirements

- Node 5.4.x
- Angular
- MongoDB

### Installing Dependencies

From within the root directory:

```sh
npm install
```
